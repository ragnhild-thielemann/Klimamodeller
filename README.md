# Modelering av klimamodeller

Når vi skal bygge den enkleste klimamodellen, anser vi jordkloden som et svart legme, og ser bort fra atmosfæren, landoverflate og havoverflate samt skyer. 

Jorden varmes opp av innomen, ultrafiolett stråling fra sola (bølgelengde på mindre enn 0.4 $\mu\$m), mens jorden reflekteter deler av strålignen og sender ut varmestråling (infrarød stråling med bølgelengde 3 - 100 $\mu\$m). 

Denne strålingen har langt lavere energitetthet enn den innkommende solstrålingen. Dette gjør at varmesstrålingen lettere absorberes av atmosfæren.

Som følge av dette blir det vanskeligere for varmestrålingen å slippe ut til verdensrommet, og energien blir delvis “fanget” i atmosfæren. Dette fenomenet er grunnlaget for drivhuseffekten, og det er grunnlaget for hvorfor jordens gjennomsnittstemperatur er mye høyere enn den ville vært uten atmosfære.
